World Explorer is a responsive React application that allows users to explore countries around the world, view detailed country information, and navigate between bordering nations using real-time data from the REST Countries API.

---

## Features

* Browse countries from around the world
* View detailed country information (population, region, capital, native name, flag)
* Navigate between **border countries**
* Fast client-side routing with React Router
* Fully responsive layout
* Styled with Tailwind CSS

---

## Tech Stack

* **Frontend:** React (Vite)
* **Routing:** React Router
* **Styling:** Tailwind CSS
* **API:** REST Countries API
* **Build Tool:** Vite


## Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/world-explorer.git

# Navigate into the project
cd world-explorer

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will run on:
`http://localhost:5173`
